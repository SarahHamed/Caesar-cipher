# Caesar-cipher

Input
The input consists of 2 lines. The first line is just a number N ( 0 ≤ N ≤ 99999999999 ) which represents the key that is also the number of right shifts. The second line is a string which represents the plain text. It consists of only capital English letters.  Each character of the string is between A-Z . No spaces, no special characters and no small letters.

Example
input:
3
HACKINGZEBRAS
output:
KDFNLQJCHEUDV
